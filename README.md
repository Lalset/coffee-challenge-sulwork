# Coffee Challenge Sulwork

Repositório para o desafio técnico: frontend Angular + backend Spring Boot + Docker.

## Estrutura
- `backend/` - API Spring Boot (Java 17, Maven)
- `frontend/` - App Angular
- `docker-compose.yml` - para levantar frontend, backend e Postgres

## Como começar (local)
1. `docker-compose up --build`
2. Backend: http://localhost:8080
3. Frontend: http://localhost:4200
